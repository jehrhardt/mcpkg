# twig

Help your coding agent twig.

## Prerequisites

`twig` uses [mise](https://mise.jdx.dev) to setup the development environment.

## Usage

Setup development environment and install dependencies:

```bash
mise install
uv sync
```

Run MCP inspector and connect to `twig`:

```bash
mise run dev:mcp
```

Run website locally:

```bash
mise run dev:page
```
